Combine two counts and display common statistical data

cancer_summary <- cancer_counts |>
  group_by(GeneId) |>
  summarise(cancer_mean_count = mean(count))

normal_summary <- normal_counts |>
  group_by(GeneId) |>
  summarise(normal_mean_count = mean(count))

merged_counts <- bind_rows(normal_counts, cancer_counts)
merged_count_summary <- merged_counts |>
  group_by(GeneId, type) |>
  summarise(mean_count = mean(count), standard_dev_count = sd(count), min_count = min(count), mx_count = max(count))

Perform T test on both datasets to compare their difference

T_test_results <- merged_counts |>
  group_by(GeneId) |>
  group_map(~ t.test(count ~ type, .x))

Gather p_value information to demonstrate the amount of data that shows statistically significant differences

p_values <- sapply(T_test_results, function(T_test) T_test$p.value)

Since we had only 19 and 23 test groups, by chance there is likely some data that shows significant differences. To reduce the number of genes that we need to analyse closely, we considered adding a second parameter： log fold of change.

The log fold of change represents the level of upregulation/downregulation by showing the order of magnitudes of mrna expression level differences.

By plotting the log of p values on the y-axis and log fold of change on the x-axis, we are able to filter through the data and select the few genes that shows both statistical significance and relatively large difference in expression level compared to normal. (top right/top left)

combine_counts_summary <- cancer_summary |>
  left_join(normal_summary) |>
  mutate(log_fold_of_change = log10(cancer_mean_count / normal_mean_count)) |>
  mutate(corrected_pvals = p.adjust(p_values, method="BH")) |>
  mutate(log_of_pvals = log10(corrected_pvals)) |>
  mutate(selected = log_of_pvals < -10.25 & log_fold_of_change > 2)
Joining with `by = join_by(GeneId)`
